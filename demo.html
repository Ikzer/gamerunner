El Game-Runner provee de un game loop de doble buffer a 6 fps, permitiendo al Juego en sí concentrarse en actualizar su estado cada vez que se llama al método update() y renderizándose a sí mismo cada vez que se llama al método draw().

El Game.Runner está pensado para ser reusable en cualquier juego JavaScript basado en canvas.

Lo sentimos, este ejemplo no se puede mostrar porque tu navegador no soporta el elemento <canvas>.
